# 개인정보처리방침

**앱:** 미국주식 공시알람  
**개발자:** LunaticWorld  
**최종 업데이트:** 2026년 4월 25일

미국주식 공시알림(이하 "서비스", "앱", "당사")은 이용자의 개인정보를 중요하게 생각합니다. 본 개인정보처리방침은 이용자가 앱을 사용할 때 어떤 정보가 수집, 이용, 공유, 보관될 수 있는지 설명합니다.

앱은 로그인 없이도 사용할 수 있습니다. 다만 계정 크레딧, 초대 기능, 계정 삭제, 계정 단위 복원 기능 등 일부 기능은 Supabase Auth를 통한 선택적 Google 로그인이 필요합니다.

## 1. 수집 또는 처리되는 정보

이용 방식에 따라 다음 정보가 수집 또는 처리될 수 있습니다.

### 1.1 로그인 시 계정 정보

이용자가 Google 계정으로 로그인하는 경우 다음 정보가 처리될 수 있습니다.

- 표시 이름
- 이메일 주소
- 사용자 ID 또는 계정 식별자
- Supabase 인증 세션 정보
- 앱 계정 설정 정보
- 계정 크레딧 잔액, 초대코드 상태, 보상 상태, 계정 삭제 상태

로그인은 선택 사항입니다. 로그인하지 않는 경우에도 기본 기능 제공을 위해 익명 세션 또는 설치 단위 식별자가 사용될 수 있습니다.

### 1.2 앱 활동 및 기능 사용 정보

서비스 제공 및 개선을 위해 다음과 같은 앱 활동 정보가 처리될 수 있습니다.

- 공시 목록 새로고침 및 공시 상세 조회
- 회사 검색 요청 및 앱 내 검색어
- 즐겨찾기 또는 관심 항목 관련 동작
- AI 요약 접근, 사용 가능 횟수, 요약 미지원 상태
- 보상형 광고 진행 상태, 보상 가능 여부, 서버 검증 상태
- 초대코드 사용 및 부정 사용 방지 확인
- 알림 주제 또는 알림 설정 상태
- 기능 사용, 화면 상호작용, 앱 안정성 확인을 위한 기본 분석 이벤트

검색어는 검색 결과 제공을 위해 사용됩니다. 검색어를 광고 프로필 생성에 사용하지 않습니다.

### 1.3 기기, 앱, 기술 정보

앱 및 서비스 제공자는 다음과 같은 기술 정보를 자동으로 처리할 수 있습니다.

- 푸시 알림 전송을 위한 Firebase Cloud Messaging 토큰
- 앱 설치 ID 또는 이와 유사한 앱 인스턴스 식별자
- 사용 가능 횟수, 보상, 부정 사용 방지에 사용되는 해시 처리된 기기 관련 식별자
- 기기 설정에서 허용되는 경우 광고 ID
- 앱 버전, 운영체제 버전, 기기 모델, 언어, 시간대, 네트워크 상태
- 진단 및 성능 정보
- 광고 또는 분석 SDK가 IP 주소를 통해 추정할 수 있는 대략적인 위치

앱은 정확한 GPS 위치 권한을 요청하지 않습니다.

### 1.4 문의 시 제공하는 정보

이용자가 이메일로 문의하는 경우, 답변을 위해 이메일 주소와 문의 내용을 처리할 수 있습니다.

## 2. 정보 이용 목적

수집 또는 처리되는 정보는 다음 목적에 사용됩니다.

- **앱 기능 제공:** SEC 공시 목록, 회사 검색, 푸시 알림, AI 요약 접근, 계정 크레딧, 초대 기능, 계정 삭제 기능 제공
- **계정 관리:** Google 로그인을 통한 Supabase Auth 인증, 계정 단위 크레딧 복원, 계정 설정 관리, 계정 삭제 처리
- **애널리틱스:** 앱 사용 방식, 기능 안정성, 성능을 파악하고 앱을 개선
- **광고 또는 마케팅:** Google AdMob 및 관련 광고 SDK를 통한 광고와 보상형 광고 제공
- **사기 예방, 보안, 규정 준수:** 보상 남용, 초대 남용, 사용 가능 횟수 우회, 잘못된 광고 보상 요청, 무단 계정 접근 방지
- **개발자 커뮤니케이션:** 이용자 문의 및 지원 요청에 응답

당사는 이용자의 개인정보를 판매하지 않습니다. 다만 앱 운영, 보안, 분석, 광고 제공을 위해 필요한 범위에서 서비스 제공자 및 SDK 파트너에게 정보가 전송 또는 공유될 수 있습니다.

앱은 다음 제3자 서비스를 사용할 수 있습니다.

| :--- | :--- | :--- |
| Supabase | 인증, 익명 세션, 계정 크레딧, 초대 기능, 요약 접근, 서버 함수, 계정 삭제 | 사용자 ID, 로그인 시 이메일 주소, 계정 상태, 크레딧 및 보상 상태, 설치/기기 기반 식별자, 앱 기능 요청 |
| Google Sign-In / Google Identity Services | 선택적 Google 로그인 | Google 계정 ID 토큰, 이름, 이메일 주소, 계정 식별자 |

앱에 표시되는 SEC 공시 정보는 공개된 원천 데이터이며 이용자의 개인정보가 아닙니다.

## 4. AI 요약 및 번역 기능

AI 요약은 공개 SEC 공시 문서를 기반으로 생성되며, 이용자의 편의를 위해 제공됩니다. 앱은 일일 조회 가능 횟수, 보상형 광고 보너스 횟수, 계정 크레딧, 부정 사용 방지 기능을 제공하기 위해 요약 접근 및 사용 가능 횟수 상태를 기록할 수 있습니다.

온디바이스 번역 기능이 활성화된 경우, 번역은 이용자의 기기에서 로컬로 수행되는 것을 의도합니다. 번역된 문구는 앱 화면 표시를 위해 사용되며, 서버의 원본 요약 데이터 자체를 변경하지 않습니다.

계정 삭제 요청이 처리되면, 합리적으로 가능한 범위에서 다음 계정 관련 정보가 삭제 또는 비활성화됩니다.

- 앱과 연결된 Supabase Auth 계정 정보
- 계정 크레딧 잔액
- 계정과 연결된 초대코드 및 보상 상태

익명 또는 로컬 전용 데이터는 일반적으로 앱 데이터를 삭제하거나 앱을 제거하면 삭제할 수 있습니다.

## 7. 데이터 보안

앱에서 서버 또는 제3자 서비스로 데이터가 전송되는 경우 HTTPS/TLS 등 암호화된 전송 방식을 사용합니다. 또한 계정 및 서비스 데이터에 대한 접근은 앱 운영, 보안, 개선에 필요한 범위로 제한합니다.

어떤 전송 또는 저장 방식도 완전히 안전하다고 보장할 수는 없지만, 당사는 서비스에서 처리되는 정보를 보호하기 위해 합리적인 조치를 취합니다.
